## 项目启动流程
#### 本地环境：
```Vue: 2.0``` `node: 16.13.2`
<!-- <span style="color:orangered;">Vue: 2.0</span> -->
> 建议使用nvm管理node版本，方便不同node版本的切换
#### 安装项目依赖
``` shell
npm install / yarn
```

#### 开发的编译和热重载
``` shell
npm run dev / yarn dev
```

#### 生产环境的编译
``` shell
npm run build:prod / yarn build:prod
```
